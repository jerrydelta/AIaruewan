自托管密码库

Self-hosted password manager run on your own server

💡 痛点

Vaultwarden 是 Bitwarden 兼容轻量自托管密码库（Rust 写，空闲内存 <50MB，树莓派可跑）。1Password 仅云、LastPass 2022-23 泄漏。隐私用户想要「自己的密码库跑在自己服务器」，自托管密码管理器渐成必备基础设施。

🛠 工具 · 本地密码库（主密码解锁）

🔒 数据仅存本机 localStorage，不上传。
